Here are some of my projects

  • Summarist Book Library

    Next.js, TypeScript, Redux, Firebase, CSS, Stripe

    The Book Library App is a feature-rich application developed using Next.js 15, TypeScript, and Firebase, designed to deliver a seamless and responsive user experience. The app leverages Redux for efficient state management and Firebase for secure authentication and real-time data updates. Key features include an interactive audio player, a customizable sidebar for navigation, and Stripe integration to handle subscription payments. This project demonstrates my ability to build modern, scalable applications with robust functionality while maintaining a user-friendly interface.

  • Ultraverse NFT Marketplace

    React, CSS, Axios, Git, GitHub

    This project involved transforming a static HTML, CSS, JavaScript, and React application into a highly interactive user interface. By introducing animations, transitions, and carousels, I enhanced the visual appeal and engagement of the application. Data was dynamically fetched from a cloud server using Axios and displayed with features like skeleton loading states, pagination, and dynamic routing. Throughout the process, I collaborated within a virtual team environment, utilizing Git version control and GitHub to manage contributions. This project strengthened my skills in modern front-end development and collaborative workflows.

  • Twitter Clone

    Next.js, Taiwind CSS, Redux

    The Twitter Clone is a modern web application built with Next.js and styled using Tailwind CSS, designed to emulate the functionality and responsiveness of a social media platform. This project features a sleek and responsive UI, enhanced by state management using Redux for dynamic modals and user interactions. By implementing real-time-like comment functionality, seamless routing, and optimized performance, this project demonstrates my ability to create interactive and efficient web applications. Through this endeavor, I deepened my expertise in full-stack development and the principles of creating engaging user experiences.
